## Point prompt format 
For point-prompt inference with SAM2, each image has a corresponding `.txt` file containing one or more point prompts.
Each line uses absolute pixel coordinates(NOT normalized ):

### Format
- `instance_id` : integer ID of the annotated instance (used to group multiple points that belong to the same object)
- `class_id` : integer class label
- `x, y` : point location in **pixels**
- `label` : point type (`1` = positive / foreground)

### Example
344 5 311 538 1
344 5 265 459 1
345 6 306 388 1
345 6 365 468 1
