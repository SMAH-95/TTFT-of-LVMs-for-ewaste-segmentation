## Box prompt format
For box-prompt inference with SAM2, each image has a corresponding `.txt` file containing one or more bounding boxes.
Each line uses absolute pixel coordinates (NOT normalized):

### Format
- `class_id` : integer class label
- `x_min, y_min` : bottom-left corner of the box in pixels
- `width, height` : box size in pixels  

### Example
5 0.00 449.00 80.00 103.12
4 34.00 34.00 262.00 269.76
6 228.00 91.00 381.19 331.75
